# Taller de Inteligencia Artificial

Este repositorio contiene la implementación de diferentes ejercicios vistos a lo largo del curso, organizados por carpetas según la semana en la que fueron desarrollados.

---

## Estructura del Taller

### Semana 3
Primer acercamiento a algoritmos de búsqueda, incluyendo métodos básicos y no informados.

- `busqueda.py`: Implementación de una **búsqueda básica**, utilizada para comprender los conceptos fundamentales.
- `breadth_first_search.py`: Implementación de **BFS (Breadth-First Search)**, que explora los nodos por niveles.
- `depth_first_search.py`: Implementación de **DFS (Depth-First Search)**, que explora en profundidad antes de retroceder.

---

### Semana 4
Introducción a algoritmos más eficientes y estrategias de optimización.

- `greedy.py`: Implementación de un algoritmo **voraz (greedy)**, que toma decisiones locales óptimas.
- `astar.py`: Implementación del algoritmo **A\***, que utiliza heurísticas para encontrar rutas óptimas de manera eficiente.
